# Cats on Unicycles

A delightful website celebrating the majestic art of cats riding unicycles. Features a gallery, history timeline, fun facts, and community submissions.

## Development

This is a static HTML/CSS/JS site. Open any HTML file directly in your browser, or use a local server:

```bash
npx live-server
```

## Structure

- `index.html` - Homepage
- `gallery.html` - Photo gallery
- `history.html` - Timeline
- `facts.html` - Fun facts
- `submit.html` - Submission form
